# 🎮 AI猜拳游戏

一个基于实时手势识别的网页版猜拳游戏，使用MediaPipe Hands进行手势检测，实现与AI的实时对战。

## ✨ 功能特点

- 🎯 **实时手势识别**：使用MediaPipe Hands技术实时检测用户手势
- 🤖 **智能AI对手**：AI会根据玩家出拳习惯采用策略性出拳
- 📊 **实时计分**：自动记录胜负和平局次数
- 🎨 **现代化UI**：美观的渐变设计和流畅的动画效果
- 📱 **响应式设计**：支持桌面和移动设备

## 🚀 快速开始

### 方法一：使用启动脚本（最简单）

```bash
./start.sh
```

### 方法二：使用 Python（推荐）

```bash
# 启动服务器
python3 -m http.server 8080

# 或者停止旧服务器后重新启动
lsof -ti:8080 | xargs kill && python3 -m http.server 8080
```

### 方法三：使用 npm（如果已安装 Node.js）

```bash
# 安装依赖（首次使用）
npm install

# 启动服务器
npm start
```

### 方法四：使用其他 HTTP 服务器

```bash
# 使用 npx http-server
npx http-server -p 8080 -o

# 或使用其他工具
php -S localhost:8080
```

然后在浏览器中打开 `http://localhost:8080`

## 🔄 重新启动项目

如果服务器已运行，需要重新启动：

```bash
# 停止当前服务器
lsof -ti:8080 | xargs kill

# 重新启动
python3 -m http.server 8080
```

或直接使用启动脚本：
```bash
./start.sh
```

## 🎮 游戏玩法

1. 点击"开始游戏"按钮启动摄像头
2. 将手放在摄像头前，做出以下手势之一：
   - ✊ **石头**：握拳
   - ✌️ **剪刀**：伸出食指和中指
   - ✋ **布**：张开手掌
3. 系统每3秒自动识别一次手势并与AI对战
4. 查看实时比分和每局结果

## 🛠️ 技术栈

- **HTML5**：页面结构
- **CSS3**：现代化样式和动画
- **JavaScript**：游戏逻辑和手势识别
- **MediaPipe Hands**：Google的手势识别技术

## 📋 手势识别规则

- **石头（Rock）**：0-1个手指伸出
- **剪刀（Scissors）**：食指和中指伸出（2个手指）
- **布（Paper）**：4-5个手指伸出

## 🎯 AI策略

AI采用混合策略：
- 70%概率随机出拳
- 30%概率根据玩家最近的手势采用克制策略

## 🌐 浏览器兼容性

- Chrome/Edge（推荐）
- Firefox
- Safari（需要HTTPS或localhost）

## 📝 注意事项

1. 需要授予浏览器摄像头访问权限
2. 确保光线充足，手势清晰可见
3. 建议在HTTPS环境下运行以获得最佳体验
4. 手势识别需要一定时间稳定，请保持手势1-2秒

## 🔧 故障排除

如果遇到"无法访问摄像头"的问题，请查看 [故障排除指南](TROUBLESHOOTING.md)

常见问题：
- **权限被拒绝**：在浏览器地址栏左侧点击摄像头图标，选择"允许"
- **macOS系统权限**：系统偏好设置 > 安全性与隐私 > 摄像头 > 允许浏览器
- **摄像头被占用**：关闭其他使用摄像头的应用（Zoom、Skype等）
